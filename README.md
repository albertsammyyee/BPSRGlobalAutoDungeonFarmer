## Initialization
   Download this project to a local path without Chinese characters  

2. Ensure you have Python 3.13.11 downloade and installed

3.
   Run the PowerShell script `Setup-Env.ps1` in the root directory to install Python dependencies  
   (If it errors, you open a Powershell Window as Administrator and run: Set-ExecutionPolicy unrestricted)
4.  
   Use `start.ps1` to start the script
